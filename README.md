# Final-Final-Web-Design-Group-Project

## Overview

This repository contains the final group project for a web design course. It includes a multi-page website built with HTML, CSS, and JavaScript, featuring a market/shop experience with product pages, sign in flows, and customer support pages.

## Contents

- `Index.html` — Home page for the website.
- `Market Folder/` — Main site content and pages for the market experience.
  - `Market.css` — Styles for the market pages.
  - `Market.js` — JavaScript interactions for the market pages.
  - `Product.html` — Product detail page.
  - `Product.css` — Styles for the product page.
  - `Product.js` — Product page behavior.
  - `Sign in.html` — Sign in form page.
  - `Sing in.js` — JavaScript for the sign in page.
  - `Forgot password.html` — Password recovery page.
  - `forgot password.css` — Styles for the forgot password page.
  - `Forgot password.js` — JavaScript for the password recovery page.
  - `Contact Us.html` — Contact form page.
  - `Contact Us.css` — Styles for the contact page.
  - `VIP.html` — VIP membership or special offers page.
  - `VIP.css` — Styles for the VIP page.
  - `Form.html` — A form page used for user interaction.
  - `Form.css` — Styles for the form page.
  - `form.js` — JavaScript for the form page.

## How to view

1. Open `Index.html` or any page in a browser.
2. If needed, use a local server to avoid browser restrictions on JavaScript and file loading.

## Notes

- The project is built with plain HTML, CSS, and JavaScript.
- File names are case-sensitive in some environments, so keep the existing file names when referencing pages.
- `Market Folder/README.md` may contain additional notes specific to the market section.
